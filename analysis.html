Cohorts

Two cohorts are used.

Echo cohort — at least one of the five core echo measurements is available (EF, lateral e′, GLS, E/A, E/e′):

Show echo-cohort filter 
echo_cohort <- master %>%
  filter(
    echo_ef_notavailable___999      == 0 |
    echo_late_notavailable___999    == 0 |
    echo_gls_notavailable___999     == 0 |
    echo_earatio_notavailable___999 == 0 |
    echo_eeratio_notavailable___999 == 0
  )

Pre-terli echo cohort — echo cohort AND the echo was done before terlipressin initiation (days_to_echo < days_to_terli_initiation_day0). Used for tasks 4–8.

Show pre-terli echo cohort filter 
preterli_echo <- echo_cohort %>%
  filter(days_to_echo < days_to_terli_initiation_day0)

Tasks 1–3 — Echo Timing Counts

Counts within the echo cohort, split by whether the echo was before, on the same day as, or after terlipressin initiation.

Show timing-count code 
timing_counts <- echo_cohort %>%
  mutate(timing = case_when(
    days_to_echo <  days_to_terli_initiation_day0 ~ "1. echo before terli",
    days_to_echo == days_to_terli_initiation_day0 ~ "2. echo same day as terli",
    days_to_echo >  days_to_terli_initiation_day0 ~ "3. echo after terli",
    TRUE                                          ~ "missing"
  )) %>%
  count(timing, name = "N")

Warningecho_gls quirk

In the pre-terli echo cohort, echo_gls ranges from −20 to +25 — some centers store GLS as a negative number (the standard convention) and others store the absolute value. 83 % of the cohort is missing GLS. Tasks 5–7 below report raw values only; medians are unstable and any future modeling on GLS should harmonize sign per center or work with abs(echo_gls).
